### Параметри запиту

<ParamField query="name" type="string">
  ПІБ фізичної особи або назва юридичної особи

  <ParamField query="ipn" type="string">
    Код фізичної або юридичної особи
  </ParamField>
  ### Параметри відповіді
</ParamField>

| Верхній рівень | Опис                                                                                                                                                                                                                                                                    |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| id             | Ідентифікатор в реєстрі                                                                                                                                                                                                                                                 |
| name           | ПІБ особи                                                                                                                                                                                                                                                               |
| rank           | Звання                                                                                                                                                                                                                                                                  |
| birth          | Дата народження                                                                                                                                                                                                                                                         |
| passport       | Паспортні дані                                                                                                                                                                                                                                                          |
| basenumber     | Номер військової частини                                                                                                                                                                                                                                                |
| base           | Назва військової частини                                                                                                                                                                                                                                                |
| info           | Додаткова інформація                                                                                                                                                                                                                                                    |
| locatedinua    | Присутність в Україні                                                                                                                                                                                                                                                   |
| is_good        | Загиблий                                                                                                                                                                                                                                                                |
| is_polon       | Полонений                                                                                                                                                                                                                                                               |
| is_identified  | Обвинувачений                                                                                                                                                                                                                                                           |
| link           | Посилання на джерело                                                                                                                                                                                                                                                    |
| address        | Адреса                                                                                                                                                                                                                                                                  |
| foto           | Фото                                                                                                                                                                                                                                                                    |
| percent        | Відсоток збігу ПІБ згідно обраного алгоритму пошуку                                                                                                                                                                                                                     |
| level          | Рівень співпадіння в реєстрі "Окупанти". Поле level може приймати два значення «Низький» і «Високий». Високий рівень співпадіння можливий лише у випадку 100% співпадіння ПІБ і дати народження. Всі решта варіанти відповіді будуть мати «Низький» рівень співпадіння. |
