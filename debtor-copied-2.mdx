### Параметри запиту

<ParamField query="name" type="string">
  ПІБ фізичної особи або назва юридичної особи

  <ParamField query="ipn" type="string">
    Код фізичної або юридичної особи
  </ParamField>
  ### Параметри відповіді
</ParamField>

| Верхній рівень | Опис                                                                                                                                                                                                                                                                                                                      |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| name           | Назва юридичної особи або ПІБ фізичної                                                                                                                                                                                                                                                                                    |
| code           | ЄДРПОУ юридичної особи або РНОКПП фізичної                                                                                                                                                                                                                                                                                |
| publisher      | Найменування органу або прізвище, ім'я, по батькові та осада посадової особи, яка видала виконавчий документ                                                                                                                                                                                                              |
| emp_fio        | ПІБ приватного виконавця                                                                                                                                                                                                                                                                                                  |
| emp_org        | Найменування органу державної виконавчої служби                                                                                                                                                                                                                                                                           |
| org_phone      | Телефони виконавця                                                                                                                                                                                                                                                                                                        |
| email          | Пошта виконавця                                                                                                                                                                                                                                                                                                           |
| vporder        | Номер виконавчого провадження                                                                                                                                                                                                                                                                                             |
| vd_cat         | Категорія стягнення (аліменти, штраф тощо)                                                                                                                                                                                                                                                                                |
| percent        | Відсоток збігу назви згідно обраного алгоритму пошуку                                                                                                                                                                                                                                                                     |
| level          | Рівень співпадіння в реєстрі Поле level може приймати два значення «Низький» і «Високий». Високий рівень співпадіння можливий лише у випадку коли значення вхідного параметру ipn (ЄДРПОУ,РНОКПП) співпадає із значенням вихідного параметру code. Всі решта варіанти відповіді будуть мати «Низький» рівень співпадіння. |
