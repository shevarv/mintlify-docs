### Параметри запиту

<ParamField query="name" type="string">
  ПІБ фізичної особи або назва юридичної особи

  <ParamField query="ipn" type="string">
    Код фізичної або юридичної особи
  </ParamField>
  ### Параметри відповіді
</ParamField>

| Верхній рівень     | Опис                                                                                                                                                                                                                 |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| id                 | Унікальний ідентифікатор запису в реєстрі                                                                                                                                                                            |
| entity             | Тип особи                                                                                                                                                                                                            |
| name               | ПІБ(назва) особи                                                                                                                                                                                                     |
| punishment_type    | Вид покарання                                                                                                                                                                                                        |
| punishment_start   | Дата набрання законної сили судовим рішенням про притягнення особи до відповідальності за корупційне правопорушення                                                                                                  |
| punishment         | Суть задоволення позовних вимог                                                                                                                                                                                      |
| sentence_date      | Дата судового рішення                                                                                                                                                                                                |
| date_disappearance | Дата зникнення                                                                                                                                                                                                       |
| offense_name       | Склад корупційного правопорушення                                                                                                                                                                                    |
| court_case_number  | Номер судової справи                                                                                                                                                                                                 |
| court_name         | Найменування суду, який ухвалив (постановив) судове рішення про притягнення особи до відповідальності за корупційне правопорушення                                                                                   |
| sentence_number    | Номер судового рішення                                                                                                                                                                                               |
| codex_articles     | Перелік статтей (розділених символом “;”), відповідно до яких особу притягнуто до відповідальності за вчинення корупційного або пов’язаного з корупцією правопорушення                                               |
| percent            | Відсоток збігу назви згідно обраного алгоритму пошуку                                                                                                                                                                |
| level              | Рівень співпадіння в реєстрі "Нотаріуси". Поле level може приймати два значення «Низький» і «Високий». Так я а даному реєстрі відсутні дата народження і РНОКПП, відповіді будуть мати «Низький» рівень співпадіння. |
