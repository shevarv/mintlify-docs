### Параметри запиту

<ParamField query="name" type="string">
  ПІБ фізичної особи або назва юридичної особи

  <ParamField query="ipn" type="string">
    Код фізичної або юридичної особи
  </ParamField>
  ### Параметри відповіді
</ParamField>

| Верхній рівень | Опис                                                                               |
| -------------- | ---------------------------------------------------------------------------------- |
| ovd            | підрозділ, яким було внесено інформацію до реєстру втрачених/викрадених документів |
| d_SERIES       | Серія документа                                                                    |
| d_NUMBER       | Номер документа                                                                    |
| d_TYPE         | Тип документа, можливі значення:                                                   |
| d_STATUS       | Статус документа, можливі значення:                                                |
| theft_DATA     | Дата повідомлення про втрату/викрадення                                            |
| insert_DATE    | Дата внесення до реєстру втрачених документів                                      |
| id             | ID запису про втрачений документ                                                   |
