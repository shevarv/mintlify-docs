### Параметри запиту

<ParamField query="name" type="string">
  ПІБ фізичної особи або назва юридичної особи

  <ParamField query="ipn" type="string">
    Код фізичної або юридичної особи
  </ParamField>
  ### Параметри відповіді
</ParamField>

| Верхній рівень | Опис                                                                                                                                                                                                                                                                 |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| code           | Ідентифікатор в реєстрі                                                                                                                                                                                                                                              |
| name           | Назва особи                                                                                                                                                                                                                                                          |
| date_reestr    | Дата реєстрації                                                                                                                                                                                                                                                      |
| percent        | Відсоток збігу назви згідно обраного алгоритму пошуку                                                                                                                                                                                                                |
| level          | Рівень співпадіння в реєстрі "Нерезеденти". Поле level може приймати два значення «Низький» і «Високий». Високий рівень співпадіння можливий лише у випадку співпадіння за кодом нерезедента. Всі решта варіанти відповіді будуть мати «Низький» рівень співпадіння. |
