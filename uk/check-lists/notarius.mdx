### Параметри запиту

<ParamField query="name" type="string">
  ПІБ фізичної особи або назва юридичної особи

  <ParamField query="ipn" type="string">
    Код фізичної або юридичної особи
  </ParamField>
  ### Параметри відповіді
</ParamField>

| Верхній рівень | Опис                                                                                                                                                                                                                 |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| name           | ПІБ нотаріуса                                                                                                                                                                                                        |
| license        | Номер ліцензії                                                                                                                                                                                                       |
| nameobj        | Юридична назва                                                                                                                                                                                                       |
| region         | Регіон                                                                                                                                                                                                               |
| contacts       | Контакти: адреса, телефони,email                                                                                                                                                                                     |
| percent        | Відсоток збігу назви згідно обраного алгоритму пошуку                                                                                                                                                                |
| level          | Рівень співпадіння в реєстрі "Нотаріуси". Поле level може приймати два значення «Низький» і «Високий». Так я а даному реєстрі відсутні дата народження і РНОКПП, відповіді будуть мати «Низький» рівень співпадіння. |
