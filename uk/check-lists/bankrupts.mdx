### Параметри запиту

<ParamField query="name" type="string">
  ПІБ фізичної особи або назва юридичної особи

  <ParamField query="ipn" type="string">
    Код фізичної або юридичної особи
  </ParamField>
  ### Параметри відповіді
</ParamField>

| Верхній рівень | Другий рівень  | Опис                                                                                                                                                                                                                 |
| -------------- | -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| edrpou         |                | ЄДРПОУ юридичної особи або РНОКПП фізичної                                                                                                                                                                           |
| name           |                | Назва юридичної особи або ПІБ фізичної                                                                                                                                                                               |
| details        |                | Детальна інформація щодо процесу банкрутства                                                                                                                                                                         |
|                | id             | ID запису                                                                                                                                                                                                            |
|                | daterec        | Дата реєстрації                                                                                                                                                                                                      |
|                | typerec        | Тип запису                                                                                                                                                                                                           |
|                | casenumber     | Номер судової справи                                                                                                                                                                                                 |
|                | startdateauc   | Дата початку аукціону                                                                                                                                                                                                |
|                | enddateauc     | Дата закінчення аукціону                                                                                                                                                                                             |
|                | courtname      | Назва суду                                                                                                                                                                                                           |
|                | endregistrdate | Дата закінчення реєстрації                                                                                                                                                                                           |
| percent        |                | Відсоток збігу назви згідно обраного алгоритму пошуку                                                                                                                                                                |
| level          |                | Рівень співпадіння в реєстрі "Нотаріуси". Поле level може приймати два значення «Низький» і «Високий». Так я а даному реєстрі відсутні дата народження і РНОКПП, відповіді будуть мати «Низький» рівень співпадіння. |
