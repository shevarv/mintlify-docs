### Параметри запиту

<ParamField query="name" type="string">
  ПІБ фізичної особи або назва юридичної особи

  <ParamField query="ipn" type="string">
    Код фізичної або юридичної особи
  </ParamField>
  ### Параметри відповіді
</ParamField>

| Верхній рівень | Опис                                                                                                                                                                                                                                                                    |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| id             | Ідентифікатор в реєстрі                                                                                                                                                                                                                                                 |
| name           | ПІБ особи                                                                                                                                                                                                                                                               |
| birth          | Рік народження                                                                                                                                                                                                                                                          |
| position       | Посада                                                                                                                                                                                                                                                                  |
| reason         | Причина по якій дана особа була додана в реєстр                                                                                                                                                                                                                         |
| foto           | Посилання на фото                                                                                                                                                                                                                                                       |
| url            | Посилання на сторінку реєстра                                                                                                                                                                                                                                           |
| source         | Посилання на джерело підтвердження факту зради                                                                                                                                                                                                                          |
| percent        | Відсоток збігу ПІБ згідно обраного алгоритму пошуку                                                                                                                                                                                                                     |
| level          | Рівень співпадіння в реєстрі "Зрадники". Поле level може приймати два значення «Низький» і «Високий». Високий рівень співпадіння можливий лише у випадку 100% співпадіння ПІБ і дати народження. Всі решта варіанти відповіді будуть мати «Низький» рівень співпадіння. |
