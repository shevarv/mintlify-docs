### Загальні питання

<AccordionGroup>
  <Accordion title="Як встановити NBFI?">
    Для роботи програми необхідно:

    1) Інсталювати JRE (Java SE Runtime Environment 8) на кожен комп’ютер, де буде виконуватися програма.

    2) Інсталювати PostgreSQL не нижче версії 9.6 (на локальний комп’ютер, або сервер).

    3) Створити (відновити, restore) базу даних “nbfitest” з необхідними довідниками та переліками даних.

    4) Розпакувати архів із програмою «CheckLists»
  </Accordion>
  <Accordion title="Як інсталювати JRE?">
    JRE (Java SE Runtime Environment 8) відповідно до вашої операційної системи можна отримати за посиланням

    https://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html

    При інсталяції варто притримуватись підказок програмного забезпечення
  </Accordion>
  <Accordion title="Як інсталювати PostgreSQL ?">
    PostgreSQL ( не нижче версії 9.6) можна отримати за посиланням:

    https://www.enterprisedb.com/downloads/postgres-postgresql-downloads
  </Accordion>
  <Accordion title="Чому програма NBFI  не запускається ?">
    Для успішного запуску необхідно:

    - перевірити версію java (перевірити версію можна натиснувши в пошуку cmd + enter);

    - якщо вища за java version "1.8.0_202", то потрібно видалити діючу версію і встановити java8.

    Алгоритм дій:

    1.Видалити поточну версію Java

    Відкрити Панель керування → Програми та компоненти

    Видалити всі встановлені версії Java

    2.Встановити потрібну версію Java (Встановіть Java 8 Update 202 (1.8.0_202))

    Використовуйте офіційний дистрибутив, рекомендований для NBFI

    3.Вимкнути автоматичне оновлення Java

    Після встановлення: відкрийте Java Control Panel/ перейдіть на вкладку Update/ зніміть галочку Check for Updates Automatically/ збережіть зміни
  </Accordion>
  <Accordion title='Якщо програма видає повідомлення "IP адресса ***.****.***.** не є в списку дозволених"'>
    Необхідно здійснити дзвінок на лінію підтримки +380980007001 (внутрішній номер 901), консультант перевірить інформацію по IP
  </Accordion>
  <Accordion title="У програмі NBFI є повідомлення «перевірка успішна», але результати перевірки не відображаються?">
    Такий результат виникає, якщо ввели неправильний логін і пароль. Логін має бути обов’язково з приставкою \*\*\*\_api
  </Accordion>
  <Accordion title="Якщо в процесі скринінгу клієнтів не по всім приходить результат?">
    Дана помилка виникає через перевищення часу очікування відповіді. У такому випадку після скринінгу усіх клієнтів буде корисним застосувати фільтр (на вкладці скринінг -\> фільтр(іконка лійочка) -\> вертикальна вкладка (дані по скринінгу) -\> ставимо галочку -кліенти, по яких не було перевірки) і примінити. Така маніпуляція покаже клієнтів, по яким, відповідь на запит перевищила час очікування і вони не перевірились. Далі процес скринінгу стандартним чином проводиться.
  </Accordion>
